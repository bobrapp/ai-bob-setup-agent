The premise

99% of legacy businesses are behind on AI. They will pay real money to catch up.

That's Nick Vasilescu's whole thesis. AI fluency is a rare, monetizable skill. Anyone who can stand up Hermes, OpenClaw, or Claude Code has leverage that marketing agencies, law firms, and manufacturers can't replicate in-house. The opportunity window is measured in months.

The catch: the work doesn't scale. Every onboarding is a 15-minute call, then an hour of manual setup — workspace, cloud computer, Hermes install, MCPs, watchdogs, Telegram channel. Do that for twenty customers and Bob is the bottleneck.

ai-bob-setup-agent is the lever. One config file. One command. Workspace + cloud computers + Hermes + MCPs + observability — provisioned. — What this repo does
What it does

One command, end-to-end onboarding.

make onboard CUSTOMER=acme-marketing takes a YAML config and produces a configured, monitored, customer-ready environment.

1

Workspace

Spins up an Orgo workspace dedicated to the customer. Scoped credentials, isolated billing.

2

Cloud computers

One VM per agent (usually 1–3). Sized per stack — 4 vCPU / 8 GB for Hermes, half that for OpenClaw.

3

Runtime install

Hermes or OpenClaw installed and licensed, base image verified, model config applied.

4

MCPs wired up

Perplexity (live search), Context7 (live docs), X (social signals). Composio + Agent Mail for actions and email.

5

Second brain

Per-customer Obsidian-style knowledge layer loaded from a seed file. Agents read and write through it.

6

Watchdog

Heartbeats every minute. Missed beat fires Telegram + email. Issues caught before the customer feels them.

The stack

Everything Nick mentioned. Wired and waiting.

LayerToolsWhy
Customer-facing Granola · Trello · Loom · Superhuman · Asana · Telegram The surface area customers actually touch. Familiar tools, no learning curve.
Agent runtime Hermes · OpenClaw · Orgo Where agents live. Orgo provides the cloud computers; Hermes / OpenClaw are the runtimes.
Connectors Composio · Agent Mail Composio gives agents app actions. Agent Mail gives them their own email addresses.
MCPs Perplexity · Context7 · X Live context. Web search, up-to-date docs, social signals.
Knowledge Obsidian (per customer) Durable second-brain layer. Agents accumulate context over time.
Models GPT 5.5 · Opus 4.7 · GLM 5.1 · Kimmy GPT 5.5 for tool calling. Opus for long-horizon coding. GLM / Kimmy for cheap volume.
Operator Codex · Claude Code Bob's coding agents. Used to write per-customer agent logic.
Verticals

Pursue the hungry. Avoid the regulated.

Per Nick: marketing, law, insurance, manufacturing, wholesale, real estate. Large legacy industries with executive pain. Healthcare and finance are out — too regulated for a solo operator.

Marketing agencies

BDR turnover, inconsistent outbound, reporting eating analyst hours. Agents replace the middle-skill labor.

Law firms

Sub-niche by practice area (matrimonial, IP, immigration). High-leverage drafting and intake work.

Insurance agencies

Quote prep, renewal nudges, claims triage. Margins love automation.

Manufacturing

Sourcing, supplier follow-up, RFQ handling. Email-heavy operations that beg for an agent.

Wholesale

Order intake, inventory questions, account-manager support. Pure throughput plays.

Real estate

Sub-niche by geography (commercial in Florida). Lead nurture, tour scheduling, doc prep.

The offer

Unlimited agents. Unlimited usage. Unlimited support.

Customers think they need ten agents. They actually need one to three. Bob prices for the imagined need and delivers against the real one. No token-talk. No credit-talk. The magic survives the contract.
