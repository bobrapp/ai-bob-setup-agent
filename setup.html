Create your Telegram Bot

The bot is how you approve/reject agent actions

  1. Open Telegram and search for @BotFather
  2. Send /newbot
  3. Name it: AIGovOps Automation
  4. Username: aigovops_foundation_bot (or any available name)
  5. BotFather will reply with your bot token
Looks like: 7123456789:AAHx1234abcd5678efgh
2

Get your Telegram Chat ID

So the bot knows where to send approval requests

  1. Open Telegram and send any message to your new bot (e.g. "hello")
  2. Open this URL in your browser (replace TOKEN with your bot token):
    https://api.telegram.org/bot[YOUR_TOKEN]/getUpdates
  3. Look for "chat":{"id":YOUR_NUMBER in the response
  4. That number is your Chat ID
A number like: 123456789 or -100123456789 (for groups)
Leave blank to use Bob's ID for both. Ken can add his later.
