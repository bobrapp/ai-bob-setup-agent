System Features

Everything the automation system does for you, organized by domain.

📧

Email Triage & Drafting

Classifies every email into 5 categories. Drafts replies for action items. Archives noise. You approve before anything sends.

● LIVE — wired to OpenAI
📅

Calendar & Meeting Prep

Proposes alternatives for conflicts. Generates pre-meeting briefings with attendee context. Extracts action items from Granola notes.

● LIVE — rule-based scheduling
🔬

AI Governance Research

Daily scan for publications on AI governance. Scores relevance to Foundation pillars. Morning digest via Telegram.

● LIVE — Perplexity + OpenAI
✍️

Content Drafting

Newsletters, LinkedIn posts (3 variants), on-demand content. Foundation voice enforced. Never publishes without approval.

● LIVE — wired to OpenAI
✅

Task & Project Tracking

Creates Asana tasks from meetings. Reminds on stale items. Syncs Trello. Weekly status reports. Outreach pipeline management.

● LIVE — Composio integration
👋

Community Welcome

Personalized DM + welcome post within 5 minutes of join. Interest tags applied automatically. Idempotent — never double-sends.

● LIVE — Circle.so + OpenAI
📰

Weekly Digest & Curation

Identifies top community posts by engagement. Drafts digest + member spotlight. Bumps high-quality inactive threads.

● LIVE — Sunday noon cycle
🛡️

Content Moderation

Classifies every post for spam, toxicity, PII, scam links. Flags instantly. Never auto-removes — you always decide.

● LIVE — real-time classification
🔔

Telegram Approval Queue

Every consequential action shows up with Approve / Reject / Edit buttons. 24h reminders. Digest mode when queue is full.

● LIVE — inline keyboards
📋

Governance & Audit

Append-only JSONL audit log. Weekly governance report. Auto-suspension on high failure rates. Full provenance trail.

● LIVE — AIGovOps rule enforced
🤝

Outreach Pipeline

Add contacts via Telegram. Auto follow-ups at 7 days. Pipeline stages tracked. Weekly status report delivered Friday.

● LIVE — Asana + LLM drafts
🔒

Dry-Run Mode

Flip one config flag and the entire system logs what it would do without making any external calls. Safe testing always.

● LIVE — one-flag toggle

Getting Started Tonight

1
Run the setup wizard

2
Test with dry-run

3
Go live

python -m src.personal_foundation — starts the bot. Approval items appear in Telegram.
