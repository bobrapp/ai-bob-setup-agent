## Sample Workflows for Demo

### Workflow 1: Email Triage (most relatable)
```
Email arrives → Agent classifies (0.8s) → "action-required, 95% confidence"
→ Agent drafts reply (2s) → Approval item in Telegram
→ Bob taps Approve → Email sent → Audit logged
Total Bob time: 3 seconds
```

### Workflow 2: New Community Member (most impressive)
```
Member joins Circle.so → Welcomer triggers (instant)
→ Reads profile: "AI governance researcher at Stanford"
→ Generates personalized DM (1.5s) → Sends automatically (pre-approved by policy)
→ Posts welcome in thread → Applies interest tags
Total Bob time: 0 seconds (policy pre-approved)
```

### Workflow 3: Weekly Newsletter (most time-saving)
```
Sunday 7 AM: Research agent has accumulated 15 high-relevance items all week
Sunday 12 PM: Curator identifies top 5 community posts
Sunday 6 PM: Writing agent assembles newsletter from both sources
→ Draft appears in Telegram with quality scorecard
→ Bob reads, taps Approve (or edits one paragraph)
→ Newsletter ready for Monday distribution
Total Bob time: 5 minutes (vs. 4 hours manually)
```

### Workflow 4: Outreach Pipeline (most systematic)
```
Bob messages bot: "outreach John Smith, CTO TechCorp, met at conference"
→ Task agent creates Asana task (pipeline: new)
→ Writing agent drafts first-contact message
→ Approval item in Telegram → Bob approves → Email sent
→ 7 days later: no response → Agent drafts follow-up → Bob approves
→ John responds "interested" → Pipeline updates automatically
Total Bob time: 2 taps (initial + follow-up)
```
